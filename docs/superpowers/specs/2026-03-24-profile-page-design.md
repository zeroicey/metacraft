# 个人中心页面设计文档

## 1. 概述

创建个人中心页面，用于用户查看和编辑个人信息，以及退出登录。

## 2. 目标

- 展示用户基本信息（头像、姓名、邮箱、简介）
- 编辑姓名和简介
- 上传头像（图片转 base64 后更新）
- 退出登录功能
- 侧边栏设置按钮跳转到个人中心页面

## 3. 设计系统

### 3.1 样式
- 使用现有 shadcn/ui 组件
- 移动端优先设计

### 3.2 响应式断点
| 断点 | 宽度 |
|------|------|
| Mobile | < 640px |
| Desktop | ≥ 640px |

## 4. 页面设计

### 4.1 布局
- 返回按钮 + 标题头部
- 头像展示区域（可点击上传）
- 用户名和邮箱展示
- 表单：姓名、简介输入框
- 保存按钮
- 退出登录按钮（底部）

### 4.2 交互
- 头像点击触发文件选择
- 图片选择后转 base64 并预览
- 保存后显示成功提示
- 退出登录清除 token 并跳转登录页

## 5. 技术实现

### 5.1 文件改动
| 操作 | 文件 |
|------|------|
| 新增 | `apps/web/src/pages/profile.tsx` |
| 修改 | `apps/web/src/router.ts` |
| 修改 | `apps/web/src/components/sidebar/index.tsx` |

### 5.2 组件
- 使用现有组件：Card, Button, Input, Textarea, Avatar
- 使用现有 hook: useCurrentUser, useUpdateUser
- 使用 auth-store 的 logout 方法

## 6. 验收标准

- [ ] 点击侧边栏设置按钮跳转到 /profile
- [ ] 页面显示用户头像、姓名、邮箱、简介
- [ ] 可以修改姓名和简介
- [ ] 可以上传头像（图片转 base64）
- [ ] 点击保存后信息更新成功
- [ ] 点击退出登录后跳转登录页
- [ ] 响应式布局正常
