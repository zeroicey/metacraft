# MetaCraft 任务列表

详细的任务文件存放在 `tasks/` 目录中。

## 任务文件编写规范

> **重要：** 任务文件应该描述 **"做什么"** 和 **"为什么"**，而不是具体代码实现细节。

### ✅ 应该包含

- **背景/目标** - 为什么要做这个任务
- **架构/设计** - 高层次的设计思路
- **涉及文件** - 需要创建/修改的文件列表
- **改动要点** - 关键改动点（表格形式，不含代码）
- **任务清单** - 可执行的 checklist

### ❌ 不应该包含

- 具体代码片段（那是执行时的事情）
- 详细的代码示例
- 具体的 API 调用方式

### 示例格式

```markdown
## 改动要点

| 改动     | 说明                |
| -------- | ------------------- |
| 图标位置 | 从 Row 开头移到结尾 |
| 无图标时 | 不显示占位符        |
```

```markdown
## 状态管理

**新增状态：**

- `currentAgent: AgentType` - 当前选中的智能体
- `yuanmengSessionId: string` - 元梦会话 ID
```
